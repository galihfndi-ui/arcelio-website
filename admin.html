Panduan Upload ke Hosting
  1. Klik "Simpan Semua" dulu

  3. Login ke cPanel / File Manager hosting kamu
  4. Navigasi ke folder website (biasanya public_html/)
  5. Upload dan timpa file lama dengan file baru
  6. Hard refresh browser (Ctrl+Shift+R) untuk melihat perubahan

Cara Kerja
  1. Kamu edit konten di admin panel ini
  2. Klik "🚀 Simpan & Deploy Sekarang"

💡 Belum punya GitHub Pages? Buat akun GitHub gratis → buat repository baru → aktifkan Pages di Settings → masukkan konfigurasi di halaman Pengaturan GitHub

⚙️ Pengaturan GitHub

Konfigurasi koneksi ke repository GitHub Pages sekali saja

Konfigurasi Repository
Buat token di: github.com/settings/tokens/new
Centang scope: repo (full control of private repositories). Token tersimpan di browser kamu saja, tidak dikirim ke mana pun selain GitHub.
Panduan Lengkap Setup GitHub Pages
  1. Daftar/login ke github.com
  2. Buat repository baru — boleh public atau private
  3. Upload semua file website (index.html, assets/, dll) ke repository
  4. Buka Settings → Pages di repository → pilih branch main → klik Save
  5. Website akan live di https://username.github.io/nama-repo/
  6. Buat Personal Access Token di Settings → Developer settings → Tokens (classic)
  7. Masukkan token + info repo ke form di atas → Simpan → Test Koneksi
  8. Setelah test ✅, pergi ke halaman Deploy ke Website dan klik Deploy!
